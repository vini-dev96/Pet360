# Pet360

## Objetivo do Site

O **Pet360** é uma landing page desenvolvida para apresentar serviços completos de cuidado e bem-estar para pets. O objetivo principal do site é:

- **Conectar tutores com profissionais qualificados** através de uma plataforma simples e acessível
- **Oferecer serviços premium** de adestramento, pet sitter, passeios monitorados e banho & tosa
- **Facilitar o agendamento** de serviços através de integração direta com WhatsApp
- **Proporcionar transparência** com relatórios, fotos e acompanhamento em tempo real dos serviços prestados
- **Garantir cuidado 360º** para pets, cobrindo desde adestramento comportamental até cuidados estéticos e de higiene

O site foi desenvolvido com foco em uma experiência de usuário moderna, responsiva e intuitiva, utilizando PHP e Tailwind CSS para criar uma interface limpa e profissional que transmite confiança e cuidado aos tutores de pets.
